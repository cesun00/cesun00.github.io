The idea of WEB-INF/ directory:

https://stackoverflow.com/questions/19786142/what-is-web-inf-used-for-in-a-java-ee-web-application

For the following structure

src
    main
        java/
            JoseServlet.java
        webapp/
            WEB-INF/
                web.xml
                hello.jsp
            index.jsp

index.jsp can be accessed directly by url, since tomcat provide the fallback JspServlet capturing all*.jsp request for you.

There is no way hello.jsp can be accessed directly by hostname/contextpath/.../hello.jsp. But hello.jsp can be referred and forwarded to from any Servlet:

1@WebServlet("/jose")
2public class JoseServlet extends HttpServlet {
3    @Override
4    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
5        System.out.println("LOGGING");
6        req.getRequestDispatcher("/WEB-INF/hello.jsp").forward(req, resp);
7    }
8}
