GNU sed stream editor

April 13, 2024
The sed program is a line-based stream editor. In the big picture, it reads a line from input, applies modification if requested by script, and print the modified line to the output, and repeats until the whole input stream is consumed. What sed Cannot sed doesn’t support Perl regex. Only POSIX BRE and ERE are available when a pattern needs to be specified; Among others, this means non-greedy matching is not possible with sed.
